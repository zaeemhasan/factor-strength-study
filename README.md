Procedure for converting a collection of raw signals into an optimally weighted combined position: 

You have N signals. Each has generated a historical return series across M time periods. The procedure converts those return series into a single optimal weight for each signal. 


STEP 1
Collect the time series of realized returns for each signal. 
For each signal i and each historical period s, record R(i,s). This is the historical profit or loss generated by acting on signal i in period s. The return series is the raw input to everything that follows.
